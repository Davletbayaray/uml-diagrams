# UML Диаграммы интернет-магазина

## 1. Диаграмма классов
Отражает структуру системы: пользователи, товары, заказы, платежи и доставка.

## 2. Диаграмма компонентов
Показывает архитектуру системы логистики и взаимодействие между компонентами.

## Использованные технологии
- PlantUML
- Draw.io
